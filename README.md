# Simple Swing App

## Introduction

~~금단의 "그 라이브러리"~~

갑자기 번개로 하루만에 개발하는 스윙 애플리케이션입니다.

## How to use

``` shell
./gradlew clean

./gradlew build
```

결과물은 `build/libs/simple-swing-app-0.0.1-SNAPSHOT.jar`에 있습니다.
